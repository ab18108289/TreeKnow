# 树知 TreeKnow - 隐私政策 / Privacy Policy

**最后更新 / Last Updated: 2026-02-10**

## 中文版

### 概述

"树知 TreeKnow"（以下简称"本扩展"）是一个浏览器扩展，用于在 AI 对话页面上捕获用户的对话内容并以思维树的形式进行本地管理。我们高度重视用户隐私。

### 数据收集

本扩展 **不会收集、上传或传输任何用户数据到外部服务器**。

具体而言：
- 本扩展不收集个人身份信息（姓名、邮箱、电话等）
- 本扩展不收集浏览历史或浏览行为数据
- 本扩展不使用任何分析服务或追踪工具
- 本扩展不包含广告

### 数据存储

本扩展捕获的对话内容和用户创建的思维树数据**仅存储在用户浏览器本地**（通过 IndexedDB 技术），不会发送到任何远程服务器。

- 所有数据完全存储在用户的设备上
- 用户可以随时通过扩展的"清空"功能删除所有数据
- 卸载扩展将自动删除所有本地存储的数据

### 权限说明

本扩展请求以下浏览器权限：

| 权限 | 用途 |
|------|------|
| `storage` | 存储用户偏好设置 |
| `activeTab` | 检测当前标签页是否为支持的 AI 平台 |
| `scripting` | 在支持的 AI 对话页面注入侧边栏界面 |
| `host_permissions`（特定 AI 平台域名） | 在豆包、DeepSeek、通义千问、Kimi、腾讯元宝、文心一言等 AI 对话页面上运行 |

### 第三方共享

本扩展 **不会与任何第三方共享用户数据**。

### 隐私政策变更

如本隐私政策发生变更，我们将在此页面更新并修改"最后更新"日期。

### 联系方式

如有隐私相关问题，请通过 GitHub Issues 联系我们。

## English Version

### Overview

"TreeKnow" (hereinafter "the Extension") is a browser extension that captures conversation content on AI chat platforms and organizes it locally as mind trees. We take user privacy very seriously.

### Data Collection

This Extension **does not collect, upload, or transmit any user data to external servers**.

Specifically:
- No personally identifiable information is collected (name, email, phone, etc.)
- No browsing history or behavior data is collected
- No analytics services or tracking tools are used
- No advertisements are included

### Data Storage

Captured conversation content and user-created mind tree data are **stored exclusively in the user's local browser** (via IndexedDB). No data is sent to any remote server.

- All data is stored entirely on the user's device
- Users can delete all data at any time via the "Clear" function
- Uninstalling the Extension automatically removes all locally stored data

### Permissions

| Permission | Purpose |
|---|---|
| `storage` | Store user preferences |
| `activeTab` | Detect if the current tab is a supported AI platform |
| `scripting` | Inject sidebar UI on supported AI chat pages |
| `host_permissions` (specific AI platform domains) | Operate on Doubao, DeepSeek, Tongyi Qianwen, Kimi, Tencent Yuanbao, Wenxin Yiyan, and other AI chat platforms |

### Third-Party Sharing

This Extension **does not share any user data with third parties**.

### Changes to This Policy

If this privacy policy is updated, we will revise the "Last Updated" date on this page.

### Contact

For privacy-related questions, please contact us via GitHub Issues.
