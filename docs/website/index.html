核心功能

你的 AI 对话，值得被记住

不再让有价值的对话消失在历史记录中，树知帮你把每次对话变成可管理的知识。

自动捕获对话

打开 AI 聊天页面即自动工作，实时捕获你的每一个提问，无需手动操作。

思维树管理

对话内容自动组织为树形结构，层次清晰，一目了然，方便回顾和梳理思路。

拖拽排序编辑

支持自由拖拽节点调整顺序，右键编辑、重命名，按你的思路重新组织。

导入导出数据

一键导出 JSON 格式数据，支持跨设备导入，你的知识永不丢失。
